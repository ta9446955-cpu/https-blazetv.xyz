[1.0.9] - 2026-07-01
Fixed
- Settings screen had blank category tabs and missing labels (Username, Password, and M3U Playlist URL weren't showing at all) because the addon had no language string file, which current Kodi requires for settings labels. Added resources/language/resource.language.en_gb/strings.po and updated settings.xml to reference it. All settings, including M3U Playlist URL, now show properly with labels.

[1.0.8] - 2026-07-01
Fixed
- The M3U Playlist setting was using a folder browser instead of a file browser, so you could never actually select an .m3u file. Now uses the correct file-picker control, filtered to .m3u/.m3u8 files (you can still switch to "All files" or type a URL directly in the dialog).

[1.0.7] - 2026-07-01
Added
- M3U Playlist URL setting is now a file browser button instead of a plain text box. You can browse to and select an .m3u file from local/downloaded storage, or still type a URL directly.

[1.0.6] - 2026-07-01
Fixed
- The "Configure" button in the addon info screen was greyed out and Settings had zero fields because settings.xml was written in Kodi's old pre-Matrix (v18 "Leia") format, which current Kodi versions don't parse. Rewrote settings.xml in the modern Kodi 19+ schema so Configure now works and shows all fields including M3U Playlist URL.

[1.0.5] - 2026-07-01
Fixed
- Settings menu wouldn't open (and every action was blocked) whenever Xtream Codes username/password were left blank, even if a valid M3U URL was set. The addon now routes normally regardless of which method you're using.

[1.0.4] - 2026-07-01
Fixed
- Added a proper "M3U Playlist URL" setting so you can paste a direct M3U link (or local file path) instead of only Xtream Codes credentials
- Fixed error when trying to load channels without Xtream Codes username/password set
- Direct M3U URL now takes priority over Xtream Codes settings when both are filled in

[1.0.0] - 2024-01-01
Initial Release
- Live channel streaming from M3U playlists
- Channel browsing by category
- Channel search functionality with fuzzy matching
- Playback history tracking
- EPG (Electronic Program Guide) support
- Configurable server credentials
- Playlist caching for offline access
- Debug logging support

Features:
✓ Browse and stream live TV channels
✓ Organize channels by category
✓ Search for channels by name
✓ Track watched channels with history
✓ View program guide information
✓ Automatic playlist caching
✓ Configurable cache duration
✓ Playback statistics

Known Limitations:
- EPG is placeholder (requires external XMLTV source integration)
- Some features may require additional configuration

Future Enhancements:
- XMLTV EPG integration
- Favorites/bookmarking
- Channel sorting options
- Subtitle support
- Recording functionality
- Advanced scheduling
- Multi-account support
