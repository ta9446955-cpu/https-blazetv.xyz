[1.0.5] - 2026-07-01
Fixed
- Settings menu wouldn't open (and every action was blocked) whenever Xtream Codes username/password were left blank, even if a valid M3U URL was set. The addon now routes normally regardless of which method you're using.

[1.0.4] - 2026-07-01
Fixed
- Added a proper "M3U Playlist URL" setting so you can paste a direct M3U link (or local file path) instead of only Xtream Codes credentials
- Fixed error when trying to load channels without Xtream Codes username/password set
- Direct M3U URL now takes priority over Xtream Codes settings when both are filled in

[1.0.0] - 2024-01-01
Initial Release
- Live channel streaming from M3U playlists
- Channel browsing by category
- Channel search functionality with fuzzy matching
- Playback history tracking
- EPG (Electronic Program Guide) support
- Configurable server credentials
- Playlist caching for offline access
- Debug logging support

Features:
✓ Browse and stream live TV channels
✓ Organize channels by category
✓ Search for channels by name
✓ Track watched channels with history
✓ View program guide information
✓ Automatic playlist caching
✓ Configurable cache duration
✓ Playback statistics

Known Limitations:
- EPG is placeholder (requires external XMLTV source integration)
- Some features may require additional configuration

Future Enhancements:
- XMLTV EPG integration
- Favorites/bookmarking
- Channel sorting options
- Subtitle support
- Recording functionality
- Advanced scheduling
- Multi-account support
