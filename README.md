# BlazeTv Kodi Repository

Official Kodi repository for **BlazeTv** - Free TV streaming addon.

## Installation

### Option 1: Install Repository (Recommended)

This allows automatic updates for the addon.

1. **Go to**: Settings > Add-ons > Install from zip file
2. **Download** the repository:
   - Direct URL: `https://raw.githubusercontent.com/ta9446955-cpu/https-blazetv.xyz/main/repository.blazetv.zip`
   
   OR manually:
   - Download: https://github.com/ta9446955-cpu/https-blazetv.xyz/tree/main/repository.blazetv
   - Zip the folder as `repository.blazetv.zip`

3. **Select** the zip file to install
4. **Restart** Kodi
5. **Go to**: Settings > Add-ons > Install from repository > BlazeTv Repository > Video Add-ons > BlazeTv
6. **Click** Install

### Option 2: Install Addon Directly

If you prefer not to use the repository:

1. **Download** the addon folder: https://github.com/ta9446955-cpu/https-blazetv.xyz/tree/main/plugin.video.blazetv
2. **Place** in Kodi addons directory:
   - Windows: `%APPDATA%\Kodi\addons\`
   - Linux/Mac: `~/.kodi/addons/`
3. **Restart** Kodi

## After Installation

1. Open the **BlazeTv** addon
2. Go to **Settings**
3. Configure:
   - Username & Password
   - Server URL (default: http://mains.services/get.php)
   - Stream Type (m3u_plus)
   - Output Format (ts)
4. Click **Live Channels** and start watching!

## Updates

The repository will notify you of addon updates automatically. Accept the update when prompted.

## Troubleshooting

### Repository won't install
- Check your internet connection
- Ensure "Unknown sources" is enabled
- Try manual addon installation instead

### Addon won't appear after repository install
- Restart Kodi completely
- Check if addon shows in Settings > Add-ons > Video add-ons
- Try reinstalling the repository
