# BlazeTv Kodi Repository

Official Kodi repository for **BlazeTv** - Free TV streaming addon.

## Downloads

| File | Direct Download |
|---|---|
| 🗂️ **Repository zip** (recommended - enables auto-updates) | [repository.blazetv.zip](https://raw.githubusercontent.com/ta9446955-cpu/https-blazetv.xyz/main/repository.blazetv/repository.blazetv.zip) |
| 📦 **Addon zip** (v1.0.5, install directly, no auto-updates) | [plugin.video.blazetv-1.0.5.zip](https://raw.githubusercontent.com/ta9446955-cpu/https-blazetv.xyz/main/plugin.video.blazetv/plugin.video.blazetv-1.0.5.zip) |

On Android/mobile, open the link in your browser to download the zip, then in Kodi use **Settings > Add-ons > Install from zip file** and pick it from your Downloads folder.

## Installation

### Option 1: Install Repository (Recommended)

This allows automatic updates for the addon.

1. **Download** the repository zip using the link above (or via `Settings > Add-ons > Install from zip file` if browsing directly from a device)
2. **Go to**: Kodi > Settings > Add-ons > Install from zip file
3. **Select** the downloaded `repository.blazetv.zip`
4. **Restart** Kodi
5. **Go to**: Settings > Add-ons > Install from repository > BlazeTv Repository > Video Add-ons > BlazeTv
6. **Click** Install

### Option 2: Install Addon Directly

If you prefer not to use the repository:

1. **Download** the addon zip using the link above
2. **Go to**: Kodi > Settings > Add-ons > Install from zip file
3. **Select** the downloaded `plugin.video.blazetv-1.0.5.zip`
4. **Restart** Kodi

## After Installation

1. Open the **BlazeTv** addon
2. Go to **Settings**
3. Configure:
   - Username & Password
   - Server URL (default: http://mains.services/get.php)
   - Stream Type (m3u_plus)
   - Output Format (ts)
4. Click **Live Channels** and start watching!

## Updates

The repository will notify you of addon updates automatically. Accept the update when prompted.

## Troubleshooting

### Repository won't install
- Check your internet connection
- Ensure "Unknown sources" is enabled
- Try manual addon installation instead

### Addon won't appear after repository install
- Restart Kodi completely
- Check if addon shows in Settings > Add-ons > Video add-ons
- Try reinstalling the repository
